 ## Architecture Recommendation
I would select Datalakehouse Architecture recommendation
Reason 1- To handle wide variety of data like structured, semi-structured, images etc. Datalakehouse can store wide variety of raw data in cost effective way giving us opportunity to query the data as well.

Reason 2- It also allows querying of the data to gain insights from data and also to build machine learning models and to perform optimization as well in more cost effective manner han data ware house.

Reason 3-datalake house can be scaled easily without losing data integrity in a cost effective manner.
